Tool to produce the data for concept extractor tool
===================================================

Run the following command from this folder. This will read the data from '../data/' and will write the output to '../input' folder

>javac -cp ;..\apache-opennlp-1.5.3-bin\apache-opennlp-1.5.3\lib\opennlp-tools-1.5.3.jar MakeData.java

Note: This presuposes that you have 'apache-opennlp-1.5.3-bin' in '../SenticNetChallengeSystem' folder
